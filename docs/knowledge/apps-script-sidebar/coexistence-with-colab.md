# 既存Colabとの共存

既存Colab/Pythonは本番一括処理、新サイドバーはHTML貼り付けの人間確認補助として共存する。

## 方針

- 既存Colab/Pythonフローは変更しない。
- MVPでは無料のLLMなし処理を標準にする。
- 高リスク変更は自動確定せず要確認候補にする。
